# movie-app-mvc
Movie app built with MVC for iOS architecture study

## Feature List
1. Load movies from the API: https://api.themoviedb.org/3/discover/movie
2. Cache response locally
3. Pagination to load all the available pages
4. Display All and Favourite movies seperately
5. Pull-to-refresh te fetch latest data from API
6. Display error when there is no data to show
7. Movie detail page with additional information(runtime, geners) from endpoint: https://api.themoviedb.org/3/movie/{movie_id}
8. Mark/Un-mark a movie as favourite
9. Navigation to and back from detail page
